# Rusty Ridge: Russian Cars Dash (Yandex Games)

Простой бесконечный аркадный клон в духе hill-racing, адаптированный под Яндекс Игры.

## Что реализовано

- Бесконечный маршрут с процедурной генерацией рельефа (без уровней).
- Случайные «аномалии» трассы (ветер, низкая гравитация, тяжёлый кузов) — это уникальная механика, которая отличает игру от обычных клонов.
- Несколько транспортных средств с разным поведением.
- Скины маршрута (базовый/закат/зима) как косметическая монетизация.
- Внутриигровая валюта (монеты), которую можно тратить на визуальные улучшения.

## Монетизация под Яндекс Игры

- Rewarded Video для ревайва после аварии (`ysdk.adv.showRewardedVideo`).
- In-App Purchase кнопка для покупки пакета монет (`payments.purchase({ id: 'coins_1000' })`).
- Косметический магазин: покупка машин и скинов за внутриигровые монеты.

> В локальном запуске (без контейнера Яндекс Игр) IAP работает в демо-режиме и просто начисляет монеты.

## Интеграция с SDK

Подключается `https://yandex.ru/games/sdk/v2`, далее при старте вызывается:

- `YaGames.init()`
- `ysdk.features.LoadingAPI.ready()`
- `ysdk.getPayments()`

## Локальный запуск

Откройте `index.html` в браузере.

Для публикации в Яндекс Играх рекомендуется упаковать проект и загрузить как HTML5-игру в кабинет разработчика.
